# デイサービス送迎運行計画アプリ

地図と連動したデイサービスの送迎ルート最適化システム

## 概要

このアプリケーションは、デイサービス事業所における送迎業務を効率化するために開発されています。利用者の住所情報をもとに、最適な巡回ルートを自動計算し、ドライバーに共有する機能を提供します。

## 主な機能

### ステップ1（現在開発中）
- ✅ 利用者・車両データの管理（Googleスプレッドシート連携）
- 🚧 今日の送迎対象者一覧表示
- 🚧 Webアプリケーション基本画面

### ステップ2（今後実装予定）
- 地図上での利用者住所の可視化
- 最適な巡回ルートの自動計算
- 移動距離・所要時間の表示

### ステップ3（今後実装予定）
- 訪問順リストの生成
- LINEへの送迎計画自動送信
- スマートフォン対応デザイン

## 技術スタック

- **フロントエンド**: HTML, CSS, JavaScript (React)
- **バックエンド**: Node.js
- **データ管理**: Google Sheets API
- **地図サービス**: Google Maps Platform（予定）
- **通知**: LINE Notify API（予定）
- **ホスティング**: Netlify / Vercel（予定）

## データ構造

### 利用者マスタ
- 利用者ID、氏名、住所、電話番号、車椅子の有無、備考

### 利用予定
- 利用者ID、利用日、希望送迎時刻、希望帰宅時刻、ステータス

### 車両マスタ
- 車両ID、車両名、定員、車椅子対応数、担当ドライバー名

### 事業所情報
- 事業所名、住所、電話番号
