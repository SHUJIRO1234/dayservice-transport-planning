## 問題の分析

### localStorageの確認
利用者マスタには正しく保存されている:
```json
{
  "name": "テスト 花子",
  "address": "東京都品川区大井1-2-3",
  "days_of_week": ["水曜日"],
  "pickup_time": "08:00"
}
```

### 考えられる原因
1. **ページリロードが必要**: カスタムイベントが発火していない可能性
2. **保存済みデータの優先**: 既に保存された送迎計画データが優先されている
3. **統合処理のタイミング**: App.jsxのuseEffectが実行されていない
